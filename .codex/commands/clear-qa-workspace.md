# /clear-qa-workspace

Очистить пользовательские и генерируемые данные QA-проекта после явного подтверждения пользователя.

Pre-flight:
- Сначала перечисли директории, которые будут очищены.
- Без явного подтверждения пользователя ничего не удаляй.
- Применяй `.codex/rules/agent-system_rules/09-clear-workspace.md`.

Выполни `.codex/prompts/agent.clear-qa-workspace.md`.
