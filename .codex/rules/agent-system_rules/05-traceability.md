# Traceability rules

- Каждый чек-лист связан минимум с одним источником трассировки: требованием, разделом ТЗ или дизайн-источником.
- Если у требования есть идентификатор, он добавляется в `Expected Result` в круглых скобках.
- Если идентификатора нет, добавляется ссылка на раздел: `(ТЗ: <раздел/пункт>)`.
- В режиме `design only` вместо требования используется название экрана или ссылка на дизайн (при наличии): `(Дизайн: <файл>/<экран>/<элемент>)`.
- Один чек-лист может ссылаться максимум на два источника трассировки.
