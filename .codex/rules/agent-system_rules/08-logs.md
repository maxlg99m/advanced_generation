# Logs rules

В директории `logs/` после выполнения пайплайна разрешены только файлы:

- `Questions.md`;
- `Mistakes.md`;
- `Details_and_Improve.md`.

Запрещено сохранять в `logs/` preview, crop, screenshots, временные изображения, промежуточные CSV, JSON, TXT или любые другие файлы.

Если для анализа большого дизайна нужны временные производные артефакты, они не должны сохраняться в `logs/`. После завершения пайплайна агент обязан проверить `logs/` и удалить собственные временные файлы, оставив только разрешенные markdown-логи.
