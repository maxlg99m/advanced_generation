# Final report rules

Краткий отчет после `/generate-qa-checklists` должен содержать:

- режим входных данных;
- платформу;
- использованные входные файлы;
- количество извлеченных требований, если есть ТЗ;
- количество итоговых чек-листов;
- статус coverage review;
- путь к CSV, если экспорт выполнен;
- открытые вопросы или блокеры, если они есть.
